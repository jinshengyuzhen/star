Hammer是什么

Hammer是将繁琐的前端开发流程打包成一套完整的工作流，然后封装在图形界面中，一键完成所有的常规功能，从而提升开发效率。

Hammer能做什么

目前Hammer可以实现“Less/Sass实时监听编译”、“Css压缩”、“雪碧图合并”、“图片压缩”、“Js压缩”、“素材上传”、“Html压缩”、“Html上传”、“浏览器同步刷新”，功能还在不断增加中。

下载安装

在安装 Hammer 之前，你需要先安装 nodejs（https://nodejs.org/）；安装完成之后，点击下方按钮下载Hammer并进行安装。

使用实例

  1. 设置用户名和邮箱地址：
    2. git config --global user.name "liyuanfeng(冯丽媛)"
    git config --global user.email "liyuanfeng@tencent.com"
  2. clone 库到本地：
    3. git clone http://git.code.oa.com/hammer/tutorial.git
  3. 安装npm包(如果失败，参考步骤8设置代理)：
    4. npm install
  4. 配置host：

    5. (为了避免referer造成的接口访问失败)
    127.0.0.1 local.qq.com

  5. 开发中：

    6. 运行gulp

    gulp

    浏览器中打开预览地址 http://local.qq.com 备注：livereload的功能需要chrome安装扩展程序(gulp-connect已升级版本，不需要安装浏览器插件)：livereload

  6. 本地构建预览：

    7. 运行gulp

    gulp build

    浏览器中打开预览地址 http://local.qq.com:8080

  7. 发布文件到线上：

    8. 运行gulp

    gulp dist

    浏览器中打开线上地址 http://sports.qq.com/ext2020/p/hammer/tutorial/index_rev.html

  8. 开发网下npm github git.oa.com代理设置：

    9. 安装npm包时，需要设置npm代理和github代理（部分npm依赖github项目）

    npm config set proxy http://proxy.tencent.com:8080
    git config --global http.proxy http://web-proxy.oa.com:8080

    注意：设置github代理后，不能访问git.oa.com的项目，需要取消github的代理

    git config --global --unset http.proxy
